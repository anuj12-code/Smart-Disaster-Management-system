Smart Disaster Management System

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) based Smart Disaster Management System designed to help users report disasters, manage emergency alerts, and monitor incidents efficiently.

This project was developed as an innovative solution for ** (SIH 2025)** under the theme of smart emergency response and disaster management.

3️⃣ Frontend Setup

📷 Project Overview

The Smart Disaster Management System enables users to:

Report disaster incidents instantly
Track disaster severity and location
Maintain centralized emergency records
Improve communication during emergencies
Support disaster response teams with structured data

This system aims to provide a scalable digital solution for efficient disaster response and emergency management.
