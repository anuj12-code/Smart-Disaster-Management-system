Smart Disaster Management System

Smart Disaster Management System is a web-based application developed to improve disaster reporting, monitoring, and emergency response management through a centralized digital platform.

The system allows users to report disaster incidents, store emergency-related information, and help authorities or organizations monitor situations more efficiently. The project focuses on creating a structured and scalable solution for handling disaster-related data and communication.

This project was developed as a solution idea for ** (SIH 2025)**.

Project Purpose

Managing disaster-related information manually can lead to delays, lack of coordination, and inefficient communication during emergencies. This project aims to simplify disaster management processes using a digital platform that supports reporting, monitoring, and data management.

The platform can be useful for:

Emergency response teams
Government authorities
NGOs and volunteers
Disaster management departments
Local administration

Installation Process
Clone Repository
git clone YOUR_GITHUB_REPOSITORY_LINK
Backend Setup
cd backend
npm install
npm start
Frontend Setup
cd frontend
npm install
npm run dev
Environment Variables

Create a .env file inside the backend folder:

MONGO_URI=your_mongodb_connection_string
PORT=5000
System Workflow

The platform allows users to:

Report disaster incidents
Store disaster-related information
Monitor affected locations
Track severity details
Maintain centralized records for emergency management

The backend handles API requests and database operations, while the frontend provides a simple interface for users and administrators.
